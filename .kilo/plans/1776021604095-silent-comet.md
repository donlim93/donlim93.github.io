# Plan: Remove remaining gaps between Tableau dashboards

## Issue
There is still visual spacing between the British Airways dashboard and Sales & Customer Analytics dashboard even after reducing the CSS gap and margin properties.

## Root Cause
There is extra spacing caused by:
1.  The embedded Tableau visualizations have fixed heights plus margin from the Tableau embed scripts
2.  The British Airways dashboard has a duplicate closing `</script>` tag at line 140 in `src/pages/tableau.astro`
3.  Additional margins may be present on the Tableau embed containers

## Proposed Fixes
1.  Remove the duplicate `</script>` closing tag in the British Airways dashboard section (line 140 of tableau.astro)
2.  Adjust Tableau container heights to remove unnecessary vertical space
3.  Add negative margins or adjust padding to eliminate remaining visual gaps
4.  Set explicit margin values on all dashboard elements to 0

## Implementation Steps
1.  Fix the duplicate closing script tag in `src/pages/tableau.astro:140`
2.  Add explicit `margin: 0` to `.dashboard-container` and `.tableauPlaceholder`
3.  Verify all dashboard sections have consistent zero bottom margins
4.  Test the page to ensure gaps are fully eliminated
